# Glossary

Glossary defines the various wordings used in context with their intended interpretation.

| Term | Meaning |
| :-:  | --- |
| Characteristics | Aspects of a system that an architecture needs to support |
| Decision | An important choice that has a long-term or significant implications for the system |
| Entity | A logical component that describes the building blocks of the systems functionality and how they interact with each other. |
| Extensibility | The ease with which a system can be extended using additional features and functionality |
| Style | build plan, overall physical shape and structure of a software system |
